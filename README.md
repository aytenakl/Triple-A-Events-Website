# Triple A Events Website

## Project Overview

A professional business website developed for Triple A, a company specialized in Events & Exhibition Management.
The website showcases the company's services, brand identity, and digital presence.

## Features

- Responsive modern design
- Premium event management landing page
- Services showcase section
- About company section
- Contact information
- Smooth scrolling animations
- Mobile-friendly layout

## Technologies Used

- HTML
- CSS
- JavaScript
